Para poder ejecutar la aplicación se require de Node.js
La ejecución de la aplicación se realiza a través de la cmd (windows) o del terminal (linux) pero es la misma para ambos.

Pasos para ejecución:
	1) Instalar node.js
	2) Desde la cmd o terminal localizar la carpeta en la que se encuentra la aplicación, en mi caso el comando para acceder a la carpeta sería: cd "C:\Users\Usuarioa\Desktop\projects\whatsapp2"
	3) Ahora ejecutamos el servidor con: npm start
	4) Para visualizar la aplicaci´´on abrimos el navegador y nos dirigimos a: localhost:3000/
